# G25 Ancestry Telegram Bot

This Telegram bot performs **ancestry analysis** using G25 datasets. It supports:

- **NNLS ancestry decomposition** using ancient populations  
- **Closest population finder** using both ancient and modern populations  
- **PCA visualization** using modern populations  

Users can **paste CSV data** or **upload CSV files** directly in the bot and receive **text results + PNG plots** in the conversation.

---

## Features

1. **NNLS Analysis**
   - Uses only ancient population data  
   - Computes contributions of ancient populations to your sample  
   - Generates pie charts for ancestry  

2. **Closest Population Finder**
   - Uses both ancient and modern populations  
   - Finds closest populations to your sample using Euclidean distance  
   - Generates bar plots of closest matches  

3. **PCA Visualization**
   - Uses modern populations  
   - Projects user sample onto PCA space  
   - Generates PCA plots  

4. **User Interaction**
   - Supports **manual paste** or **file upload**  
   - Provides **population suggestions** when typing names
